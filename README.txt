This is simple Moodle block plugin based of selfenrol plugin which will shows the remaining and total seats available in course.

==block settings ==
* Block title
* Block Content
* Hide block header
* Selfenrol instance

== place holders in block content setting == 
* add [[coursename]]  
* add [[remaining_seats]]
* add [[total_seats]]


* Full html tag supported in block content setting.

Note: This block is created as per the following requirement https://moodle.org/mod/forum/discuss.php?d=215003
